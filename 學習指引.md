# 北市大地球科學課程學習指引

## 📚 課程概述

**課程名稱**: 北市大地球科學課程計劃  
**主題**: 地震學、地球物理學與 AI/Agent 輔助教學  
**課程期間**: 6週（2026/7/17 - 2026/8/21）  
**授課對象**: 中小學教師  

---

## 🎯 學習目標

### 核心目標
1. ✅ **建立基礎知識** - 打好地震學和地球物理學的基礎
2. ✅ **實作能力** - 學會使用各種地震資料進行簡單分析和教學設計
3. ✅ **工具熟練** - 熟悉 opencode、Antigravity CLI、Hermes agent、GitHub、Hugging Face、Vercel、Google GAS、Google GWS 等工具
4. ✅ **教學應用** - 將地震學概念轉化為中小學可理解的教學活動
5. ✅ **自動化流程** - 透過 Google GWS 與 GAS 建立教學自動化、資料收集與成果展示流程
6. ✅ **每週成果** - 每週完成一個可評量的作業
7. ✅ **野外實作** - 安排一次野外實地觀測或簡易地震百測活動
8. ✅ **最終成果** - 產出一份可帶回學校使用的地震教學作品

---

## 📅 6週課程內容與學習重點

### 🔹 Week 1: 課程介紹、平台註冊、環境建置與工作流程建立
**日期**: 2026/7/17

#### 學習重點
- [ ] 課程概述與學習目標了解
- [ ] 各平台註冊與環境設定
- [ ] 工作流程建立

#### 具體任務
1. **平台註冊**
   - [ ] Google Drive 課程資料夾
   - [ ] GitHub 個人帳號與 repo
   - [ ] Hugging Face 帳號
   - [ ] Vercel 帳號

2. **工具安裝**
   - [ ] Git
   - [ ] VS Code
   - [ ] Python
   - [ ] Node.js
   - [ ] Java

3. **AI/Agent 工具**
   - [ ] opencode
   - [ ] Antigravity CLI
   - [ ] Hermes agent

4. **Google 工具**
   - [ ] Google Forms 測試表單
   - [ ] Google Sheets 回應表
   - [ ] Google Apps Script (GAS) 測試程式

5. **資源探索**
   - [ ] 瀏覽 IRIS InClass 地震教學資源
   - [ ] 制定個人學習目標

#### 作業 1: 完成個人工作平台建置
**繳交內容**:
1. Google Drive 個人課程資料夾截圖
2. GitHub 個人課程 repo 連結
3. Hugging Face 個人頁面截圖或連結
4. Vercel 個人頁面截圖

6. Google Sheets 回應表截圖
7. GAS 測試成功截圖

- 平台: Google、GitHub、Hugging Face、Vercel 至少完成 3 項
- GitHub: 有建立 repo，並包含 README.md
- Google GAS: 能成功執行一段測試程式
- 安裝: Git、VS Code、Python 至少完成 2 項
- 學習目標: 能清楚說明自己希望把哪些地震概念轉化為教學活動

---

### 🔹 Week 2: 地震強、加速度、地震動與震級
**日期**: 2026/7/24

#### 學習重點
- [ ] 地震強度的基本概念
- [ ] 加速度與震動的關係
- [ ] 震級的定義與計算
- [ ] 地震動的物理意義

#### 具體任務
1. **理論學習**
   - 了解地震強度的記錄方式
   - 理解加速度、速度、位移的關係
   - 學習震級的計算方法

2. **實作練習**
   - [ ] 使用手機感測器、Google Sheets、GAS、IRIS ground-shaking 活動
   - 進行地震動資料分析

#### 作業 2: 完成簡易震動測量活動
**繳交內容**:
1. 一張手機或感測器震動紀錄
2. 一張 Google Sheets 圖表
3. 一份教師版教學簡報
4. 一份學生版教學簡報
5. 200 字教學心得: 這些活動如何幫助學生理解震級？

**評量標準**:
- 資料: 至少有一張實測或模擬震動資料
- 圖表: 能呈現震動強度變化
- 教師版: 至少包含 3 個教學重點
- 學生版: 至少包含 2 個學生需要理解的概念
- 教學心得: 能清楚說明活動的教學價值

---

### 🔹 Week 3: P-S 到時差、旅行時間曲線與震央定位
**日期**: 2026/7/31

#### 學習重點
- [ ] P 波與 S 波的特性
- [ ] 到時差的概念
- [ ] 旅行時間曲線的繪製
- [ ] 震央定位的方法

#### 具體任務
1. **理論學習**
   - 了解 P 波和 S 波的傳播特性
   - 學習到時差的計算方法
   - 理解旅行時間曲線的意義

2. **實作練習**
   - [ ] 使用 IRIS 震央定位活動
   - 使用 Python、opencode、GitHub
   - 進行震央定位計算

#### 作業 3: 完成震央定位教學範例
**繳交內容**:
1. 三張測站的 P 波與 S 波到時表
2. P-S 到時差計算表
3. 震央位置計算結果
4. 三測站定位圖
5. GitHub 程式或 Google Sheets 連結
6. 一份教學提問設計

**評量標準**:
- P/S 到時: 能正確標示至少九張測站資料
- 震央位置: 能由 P-S 到時差推估震央位置
- 定位圖: 能繪製三測站定位圖
- 教學提問: 至少需要九個測站

---

### 🔹 Week 4: 課本第 4 章精讀與地震波形資料
**日期**: 2026/8/7

#### 學習重點
- [ ] 地震波的基本概念
- [ ] 波形分析的方法
- [ ] 地震儀的工作原理
- [ ] 地震波形的解讀

#### 具體任務
1. **理論學習**
   - 閱讀課本第 4 章
   - 理解地震波的基本概念
   - 學習波形分析的方法

2. **實作練習**
   - [ ] 使用 jAmaSeis 或其他地震波形工具
   - 操作真實地震事件資料
   - 進行波形分析與教學轉化

#### 作業 4: 完成第 4 章導讀與波形分析小報告
**繳交內容**:
1. 一段第 4 章中文導讀
2. 一張真實地震波形圖或截圖
3. 標示 P 波、S 波或其他重要波形
4. 三個教學重點整理
5. 一段 200 字反思: 哪些內容適合中小學？

**評量標準**:
- 導讀: 能用教師可理解的方式解釋第 4 章重點內容
- 波形圖: 至少有一張真實或模擬波形圖
- 標示: 能標示 P 波、S 波或其他重要波形
- 教學重點: 至少整理出 3 個教學重點
- 反思: 能說明哪些內容適合中小學教學

---

## 💡 學習建議

### 1. 事前準備
- **工具安裝**: 提前安裝所有需要的軟體和工具
- **帳號註冊**: 提前註冊所有平台的帳號
- **基礎知識**: 閱讀相關的地震學基礎知識

### 2. 課堂參與
- **積極提問**: 不懂的地方及時提問
- **記筆記**: 記錄重要的概念和操作步驟
- **小組討論**: 與同學討論和分享經驗

### 3. 實作練習
- **跟著做**: 按照老師的指導進行操作
- **多練習**: 利用課餘時間多練習
- **記錄問題**: 遇到的問題和解決方法

### 4. 作業完成
- **按時繳交**: 按時完成並繳交作業
- **質量為先**: 注重作業的質量而非數量
- **互相學習**: 參考同學的優秀作業

### 5. 反思改進
- **課後反思**: 每週課後進行反思
- **記錄心得**: 記錄學習心得和改進點
- **持續學習**: 課程結束後繼續學習和實踐
